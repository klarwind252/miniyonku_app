# ミニ四駆レース管理システム

ミニ四駆レースの運営を一元管理する Web アプリケーションです。エントリー管理・予選スケジュール生成・決勝トーナメント管理・表彰状作成・観覧画面の配信・料金計算などを、パソコン1台（クラウド版はVPS1台）で行えます。

- **著作権フリー**（パブリックドメイン / The Unlicense）— 利用・改変・再配布・商用利用すべて自由。許諾・クレジット表記・連絡は不要です。
- **サポートなし**（原則）— 各自の責任でご利用ください。
- **オンプレ版**（**Windows PC / Mac** 1台）と、**クラウド版**（VPS常設・スマホ観覧配信対応）の2形態。

> **★ この版での更新（2026年7月・改訂）**
> - オンプレ版が **macOS に対応**しました（`.command` をダブルクリックで導入・起動）。
> - 起動スクリプト（`start.bat` / `start.command`）を **自己位置解決**にし、OneDrive / iCloud 配下や任意のフォルダでも動くようにしました（「解凍した場所でそのまま動く」run-in-place 方式）。
> - オンプレ版の待ち受けを既定で **localhost 限定（127.0.0.1）** にしました。LAN の観覧端末に見せる場合のみ `0.0.0.0` に切り替えます。
> - 配布物から `desktop.ini` 等の不要ファイルを除去し、`.gitignore` に `*.zip` を追加しました。

## 主な機能

- エントリー管理（QR / バーコード / NFC のスキャン受付）
- 複数の予選形式（並び順（ポイント制）／並び順（勝ち抜け）／ヒート（トーナメント・総当たり）／ポイント制／総当たり／即決勝 など）
- 決勝トーナメントの自動生成（シード3段階・敗者復活トーナメント対応）
- 表彰状の自動生成（PDF）／ X（旧Twitter）へのリザルトポスト文の自動生成
- 参加者向け観覧画面の配信（クラウド版はスマホでリアルタイム閲覧）
- 料金計算（曜日・祝日別、大人／子供別）
- PWA ホーム画面アイコン・アクセス QR の生成
- 複数店舗展開・お試し（デモ）予約（クラウド版）

詳しい仕様は、リポジトリ同梱の **`仕様書.docx`** を参照してください。

## 動作環境

- **オンプレ版（Windows）**：Windows 10 (64bit) / Windows 11、Python 3.12（同梱インストーラあり）
- **オンプレ版（Mac）**：macOS（Apple Silicon / Intel）、Python 3.12（universal2 版・オンライン導入）
- **クラウド版**：Ubuntu 24.04 LTS（Python 3.12）／ nginx 前段 ／ VPS 月額1,000円前後〜

## セットアップ

導入手順はリポジトリ同梱の Word ファイルにまとめています。

| 目的 | 参照ファイル |
|------|------|
| まず読む | `はじめにお読みください.docx` |
| オンプレ版の導入（Windows / Mac） | `オンプレ版_インストール手順書.docx` |
| オンプレ版の更新 | `オンプレ版_アップデート手順書.docx` |
| クラウド版の構築 | `クラウド版_インストール手順書.docx` |
| 全体の仕様 | `仕様書.docx` |

### かんたん導入（Mac）

1. フォルダを任意の場所（推奨：`~/miniyonku_app`）に解凍する
2. `setup/setup_mac.command` をダブルクリック（初回のみ・要ネット）
3. `start/start.command` をダブルクリックで起動 → 自動でブラウザが開く

### かんたん導入（Windows）

1. フォルダを解凍する
2. `setup\初回セットアップ・レストア用.bat` を実行
3. `start\start.bat`（または `Mini4wd.exe`）で起動

## リポジトリ構成

```
miniyonku_app/
├── app/        アプリ本体（FastAPI）。内部はクリーンアーキテクチャで分離：
│   ├── core/           設定・店舗コンテキスト等の基盤
│   ├── domain/         純粋なドメイン知識（判定・ラベル・締切・かな 等）
│   ├── application/    ユースケース（サービス層）
│   ├── infrastructure/ DB接続・スキーマ・リポジトリ
│   ├── presentation/   ルーター・認証・ミドルウェア・テンプレート
│   ├── templates/      Jinja2 テンプレート
│   └── static/         静的ファイル
├── deploy/     クラウド版デプロイ用ファイル（nginx・systemd・環境変数の見本）
├── setup/      セットアップ用スクリプト・依存パッケージ
│   ├── 初回セットアップ・レストア用.bat   （Windows）
│   ├── setup_mac.command                  （Mac）
│   ├── requirements.txt / requirements-mac.txt
│   └── …（各種初期化・migrate の .bat / .command）
├── start/      起動用スクリプト（start.bat / start.command）
├── *.docx      各種手順書・仕様書
├── .gitignore
└── LICENSE
```

> **注意：** 本番トークンを含む `deploy/miniyonku.env` や接続情報の控え `setup/アクセス用.txt` は、`.gitignore` で除外しており、このリポジトリには**含まれません**。各自の環境で見本ファイル（`deploy/.env.example` 等）から作成してください。ビルド成果物の `*.zip` も `.gitignore` で除外しています。
